# Credits

`codesieve` was inspired in part by [`jcodemunch-mcp`](https://github.com/jgravelle/jcodemunch-mcp), which helped highlight the value of token-efficient, structured code retrieval for AI assistants.

This project is intended as an independent, from-scratch implementation with its own architecture and goals, including a Go-based CLI-first design, SQLite-backed indexing, and optional semantic/LSP enrichment.

Many thanks to the original project for helping frame the problem space and demonstrate why retrieving precise code context can be more effective than brute-force file loading.
